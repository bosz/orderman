Brief about

Orderman is a very simple order tracking system developed using HTML, CSS, JavaScript, Jquery and Twitter Bootstrap and PHP

How it works

  1. With order man, the user to add new order items to the list. The data is submitted via an ajax call and stored in a json file.

  2. To ensure the user knows what's in his/her json db, each time the user reloads the screen, the data is loaded from the json file through ajax.
